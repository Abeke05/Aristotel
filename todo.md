# Python Streamlit Университетская Система - План разработки

## Цель
Переделать React/TypeScript версию университетского сайта в Python Streamlit приложение

## Файлы для создания

### 1. app.py - Главное Streamlit приложение
- Система авторизации и навигации
- Роутинг между страницами студента и преподавателя
- Управление сессиями

### 2. models.py - Модели данных
- Класс User (студент/преподаватель)
- Класс Grade (оценки)
- Класс Schedule (расписание)

### 3. database.py - Управление данными
- Функции для работы с JSON файлами
- CRUD операции для пользователей, оценок, расписания
- Инициализация демо данных

### 4. pages/student_dashboard.py - Панель студента
- Просмотр оценок
- Просмотр расписания
- Профиль студента

### 5. pages/teacher_dashboard.py - Панель преподавателя
- Управление оценками студентов
- Управление расписанием
- Список студентов

### 6. requirements.txt - Зависимости Python
- streamlit
- pandas
- json

### 7. data/ - Папка с данными
- users.json - пользователи
- grades.json - оценки
- schedule.json - расписание

## Особенности реализации
- Использование st.session_state для управления авторизацией
- JSON файлы для хранения данных (аналог localStorage)
- Современный дизайн с Streamlit компонентами
- Русский интерфейс
- Демо данные для быстрого тестирования
